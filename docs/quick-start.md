# Quick Start

## Install

::: code-group

```sh [npm]
$ npm install -D demo
```

```sh [pnpm]
$ pnpm add -D demo@latest
```

```sh [yarn]
$ yarn add -D demo
```

:::

## Runtime API

### add

Calculates the result of adding two numbers

```js
add(1, 2) // 3
```

### reduce

Computes the result of subtracting two numbers

```js
add(2, 1) // 1
```
